# 2023년 신문윤리위원회 심의자료 분석 결과

## 새로운 평가기준 패턴 발굴

### 분석 개요

- **분석 대상**: 2023년 신문윤리위원회 심의자료 (약 2,500건 추정)
- **분석 목적**: CR 프로젝트 기존 8가지 평가기준에 포함되지 않은 새로운 문제적 보도 패턴 발굴
- **분석 방법**: 기존 평가기준과 대조하여 새롭게 나타나는 위반 유형 도출

### 주요 발견 패턴

## 1. 자극적 범죄보도의 세부 묘사

### 패턴 특징

- 흉기 사용 범죄에서 무기 종류(커터칼, 낫, 손도끼)와 행위('휘두른', '내리친')를 구체적으로 묘사
- 학교폭력 가해 장면을 그래픽이나 연출 사진으로 시각화
- 마약 사용 장면이나 도구를 노골적으로 노출

### 실제 사례

- "커터칼 휘두른 60대", "낫 휘두른 70대", "손도끼로 손등 내리친 20대" 등 제목 사용
- 3살 아기가 필로폰 파이프를 물고 있는 사진 게재
- 학교폭력 가해 장면을 그래픽으로 구체적 묘사 (멱살잡기, 무릎꿇리기, 발길질 등)
- 화살이 개의 몸을 관통한 장면 무편집 게재

### 기존 기준과의 차이점

- 기존 '선정보도 금지' 조항이 있으나, 범죄 도구와 행위의 구체적 묘사에 대한 세부 지침 부재
- 청소년 유해환경 보호 관련 조항이 있으나, 시각적 폭력 묘사에 대한 구체적 기준 미흡

### 추가 권장사항

**카테고리**: 1-7. 언어와 표현의 윤리
**하위 항목**: 1-7-3. 자극적 표현 (보강)

- 범죄 도구의 구체적 명칭 사용 자제
- 폭력 행위의 세부적 묘사 금지
- 범죄 재현 가능성이 있는 시각자료 사용 제한

## 2. 음주운전 시각화 문제

### 패턴 특징

- 음주운전의 위험성을 알리려는 의도지만 오히려 위험한 행위를 시각화

### 실제 사례

- 대낮 스쿨존 음주운전 사고 기사에 술병 든 운전자 사진 게재
- 음주운전 단속 기사에 운전 중 음주 장면 연출

### 기존 기준과의 차이점

- '선정보도 금지'에 포함될 수 있으나, 위험 행위 연출에 대한 명확한 지침 없음

### 추가 권장사항

**하위 항목**: 1-8-4. 위험 행위 시각화 금지

- 음주운전, 약물 사용 등 불법 행위 연출 사진 사용 금지

## 3. 이미지 저작권 침해의 체계적 패턴

### 패턴 특징

- 게티이미지, 셔터스톡, 아이스톡 등 유료 이미지 사이트 자료 무단 사용
- 온라인판에는 출처 표기하면서 지면에는 표기 누락
- 자체 그래픽 작업 후에도 원저작권 표시 없음

### 실제 사례

- "여름 자외선에 지친 내 얼굴" 기사에 게티이미지 사진 무단 사용
- "반려견 감기주의보" 기사에 셔터스톡 이미지 출처 미표기
- 원본 이미지 일부 수정 후 출처 표기 생략

### 기존 기준과의 차이점

- '저작물의 전재와 인용' 조항 존재하나, 상업용 이미지 사이트 관련 구체적 지침 부재

### 추가 권장사항

**카테고리**: 1-8. 디지털 환경의 윤리
**하위 항목**: 1-8-3. 디지털 플랫폼 특유 문제 (보강)

- 상업용 이미지 데이터베이스 사용 시 출처 명시 의무화
- 원본 이미지 수정 시에도 원저작권자 표시 필수
- 온라인과 지면 모두 동일한 저작권 표시 기준 적용

## 4. 제목과 본문의 심각한 불일치

### 패턴 특징

- 인용부호를 사용한 직접 인용 제목이지만 실제 발언이 없는 경우
- 제3자 해석을 당사자 발언처럼 제목에 사용

### 실제 사례

- "尹, 국민만 보고 '파업 악순환' 끊어야" - 실제로 윤 대통령이 한 발언 아님
- "한국지사장? 안가요" - 구체적 발언자나 출처 없이 일반화

### 기존 기준과의 차이점

- '편집지침-제목의 원칙' 위반이나, 인용부호 오용에 대한 세부 지침 필요

### 추가 권장사항

**카테고리**: 1-1. 진실성과 정확성
**하위 항목**: 1-1-4. 사실과 의견 혼재 (보강)

- 직접 인용 제목 사용 시 실제 발언 확인 의무화
- 해석이나 요약은 인용부호 사용 금지
- 제3자 전언은 간접 화법으로 표현

## 5. 선거보도의 특수한 문제

- 조사기관의 계통적 편향성 무시

### 실제 사례

- 심의자료에는 구체적 사례가 제한적이나, 데이터 왜곡 부분에서 언급

### 기존 기준과의 차이점

- '데이터 및 통계 오용'에 포함되나, 선거보도 특수성 미반영

### 추가 권장사항

**카테고리**: 1-1. 진실성과 정확성
**하위 항목**: 1-1-5. 데이터 및 통계 오용 (보강)

- 선거 여론조사 보도 특별 지침 추가
- 오차범위 의무 표기 및 해석 기준 명시
